## Overview

Ce composant permet de créer des éléments cliquables au sein de votre application.
Par exemple, avec ce bouton vous pouvez soumettre des formulaires, ouvrir des popins, exécuter une action, ou renvoyer vers une autre page.

<Canvas>

        setup() {
          return { args }
        },
        template: `<BaseButton v-bind="args">{{ args.default }}</BaseButton>`,
      }
    }}
  </Story>
</Canvas>

## Usage

Le composant peut être utiliser dès le moment où l'on souhaite construire une intéraction cliquable avec l'utilisateur.

Dans le cas où c'est un lien, il n'est pas nécessaire d'utiliser un bouton sauf si l'on souhaite conserver le rendu style "bouton".
Cependant, si vous souhaitez créer un lien avec un rendu de style "bouton",
vous pouvez spécifier la propriété `href` pour un lien classique ou `to` qui utilisera le `RouterLink` de [VueRouter](https://router.vuejs.org/).
La sémantique du composant s'adapte correctement pour changer le tag HTML "a" ou "button" en fonction de son utilisation pour une meilleure logique sémantique.

Attention, le composant est de base de type `button`, contrairement à l'élement HTML qui, de base, est à `submit`.
Le choix est que par défaut un bouton est une interaction classique et que lorsque l'on choisit que cette interaction est une soumission de formulaire,
il est obligatoire de rajouter `type="ButtonType.SUBMIT"` au composant.

## Variants

### Themes

La propriété `theme` donne accès à différents styles de bouton. Il permet aussi de créer une hiérarchie des actions sur l'application.

<Canvas>
  <Story name="Theme">
    {() => {
      return {
        components: { BaseButton },
        template: Object.values(ButtonTheme)
          .map(
            (theme) =>
              `<BaseButton theme="${theme}">
                ${theme ? theme : 'default'}
              </BaseButton>`
          )
          .join(' '),
      }
    }}
  </Story>
</Canvas>

### Sizes

La propriété `size` donne accès à différents tailles de bouton pour convenir à différents scénarios.

<Canvas>
  <Story name="Size">
    {() => {
      return {
        components: { BaseButton },
        template: Object.values(ButtonSize)
          .map(
            (size) =>
              `<BaseButton size="${size}">
                ${size}
              </BaseButton>`
          )
          .join(' '),
      }
    }}
  </Story>
</Canvas>

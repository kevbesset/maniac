## Overview

Maniac est équipé d'un outil de gestion de thème qui permet de personnaliser l'apparence de l'application, et de la changer facilement.
Cette gestion se situe dans le composant `ThemeManager.vue` de votre projet.

À noter que dans cette documentation, le composant est installé sur tous les autres composants,
et vous pouver le manipuler en modifiant la valeur dans la toolbar en haut de votre écran "Theme" pour voir comment rendre ces composants avec un thème différent.

## Comment créer un nouveau thème

Pour créer un nouveau thème, il faut d'abord l'ajouter dans la liste des thèmes disponibles dans `ThemeName` qui se trouve dans `vars/ThemeAttr.ts`.
Ensuite, il faut bien sûr gérer les nouvelles couleurs dans votre `assets/sass/_theme.scss` en ajoutant votre modifier BEM `&--new-theme` et surcharger uniquement les couleurs que vous souhaitez changer.

Vous ne devez pas créer de nouvelles variables dans un thème car celle-ci doit forcément avoir un équivalent dans le thème par défaut.

Enfin, vous pouvez bien sûr ajouter autant de variables CSS que vous le souhaitez dans le thème par défaut, et ensuite les surcharger dans le thème que vous souhaitez.

## Usage

Le composant est pensé pour être utilisé globalement dans votre projet.
Il est donc conseillé de le rajouter dans le fichier `App.vue` et entouré l'entièreté de votre code car uniquement ce qui sera à l'intérieur de ce composant disposera des variables CSS.

```vue
<ThemeManager>
    ... contenu de votre application
</ThemeManager>
```

Ensuite, si vous souhaitez changer le thème dans votre application, vous pouvez le faire en accédant aux données que ce composant [provide](https://vuejs.org/guide/components/provide-inject.html).

```vue
import { inject, Ref, ref } from 'vue'
import { ThemeName, ThemeProvider } from '@/vars/ThemeAttr'

const theme = inject<Readonly<Ref<ThemeName>>>(ThemeProvider.THEME)
const setTheme = inject<(theme: ThemeName) => void>(ThemeProvider.SET_THEME)

function toggleTheme() {
    if (typeof setTheme !== 'undefined' && typeof theme !== 'undefined') {
        setTheme(
            theme.value === ThemeName.DEFAULT ? ThemeName.DARK : ThemeName.DEFAULT
        )
    }
}
```
